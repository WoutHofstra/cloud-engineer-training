# project1-Azure-Lab

## Project overview
This project is my first project using Terraform and Azure DevOps. The goal is to make a lab environment in Azure that gets deployed automatically on Saturday morning, and gets destroyed automatically on Saturday evening. That way I don't have to set up a lab environment manually every time I want to practice things in Azure. Also that would give me time every week to modify the Terraform code if I want to try something different or if I want to change anything.  

Additionally, this project will help me discover and practice different ways to make a secure environment in Azure. I will try to configure this project in different ways (referred to as versions in here), where all of them are secure. In future projects this experience will help me focus on different things, like AKS or monitoring, instead of having to focus a lot on the networking aspect that any project in Azure will have.

Also this will help me learn some basics about Terraform and Azure DevOps. With this basis I will be able to practice more difficult things like working with AKS, or maybe write some Python to automate some extra things.
