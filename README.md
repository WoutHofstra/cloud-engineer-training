# cloud-engineer-training

## A repo where I document my training to become a cloud engineer

My deadline in general is March 2027. By then I will want to have learned all the skills I think are necessary to become a cloud engineer. 

## What will I achieve?

By March 2027, I will have: 
* Achieved the AZ-104 certiticate
* Deployed IaC in Azure using Terraform, Python and GitHub
* Achieved the Terraform Associate 004 certificate
* Done an project involving AKS
* Achieved the Kubernetes and Cloud Native Associate (KCNA) certificate
* Learned about DataDog

## Why these skills?

I have chosen these skills because I see that they are quite common in the Cloud Engineering field. Also, I have prioritised learning software/ skills that actually interest me. If I am not interested in what I am learning, I will not be motivated and will not get to the level I want to reach.
